Проект: простой статический сайт с фоновой картинкой и музыкой

Что делает:
- Показывает фон (поставьте вашу картинку в `assets/bg.jpg`).
- В центре показывает ник `hraine_IVANTUS`.
- Пытается автоматически воспроизвести локальный файл `assets/unravel.mp3` при загрузке.
- Если автоплей блокируется, отображается кнопка для ручного запуска.
- Опционально можно использовать встраивание YouTube: укажите ID видео в `ytVideoId` в `index.html`.

Как использовать:
1) Положите ваш фон `bg.jpg` в папку `assets` и аудиофайл `unravel.mp3` (если у вас есть лицензия на использование).
2) Откройте `j:\copilot\anime-site\index.html` в браузере (двойной клик или `Start-Process` в PowerShell).

Важно по авторским правам:
- Я не добавляю сам файл с музыкой. Чтобы воспроизвести "Tokyo Ghoul - opening 1", используйте собственный лицензионный файл `assets/unravel.mp3` или встраивайте легальный YouTube-клип, указав его ID.
- Автоплей со звуком часто блокируется браузерами; если звук не стартует автоматически, нажмите кнопку воспроизведения.

Если хотите, могу автоматически вставить конкретный YouTube video ID (если у вас ссылка) или подключить сетевой URL с аудио.
